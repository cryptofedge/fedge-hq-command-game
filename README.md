# FEDGE HQ Command Game

A private, single-file FEDGE 2.O command game prototype.

The experience includes:

- A modern FEDGE HQ office map
- Agent offices and character avatars
- A meeting room with standup flow
- Mission deck, shards, focus, and build scoring
- HQ Reels for short-form agent progress updates
- GitHub repo cards and private access sections

## Run Locally

Open the HTML file directly, or run the included local preview server:

```bash
node static-preview-server.cjs 8765
```

Then visit:

```text
http://127.0.0.1:8765/fedge2o-command-center-whop-style.html
```

## Files

- `fedge2o-command-center-whop-style.html` - main playable prototype
- `static-preview-server.cjs` - tiny local static server
